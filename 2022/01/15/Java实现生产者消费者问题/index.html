生产者消费者synchronized版实现

为什么使用 while 而不是使用 if ？

因为多线程中可能存在虚假唤醒问题，如果用 if 那当遇到虚假唤醒时直接停止 wait 进入到下一步程序中，但如果用 while 遇到虚假唤醒时程序会再次对 resource 进行判断资源条件是否满足以防止程序错误地进入下一步执行。
